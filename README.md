# PoolGame
3D 당구 시뮬레이션

2003년 여름 LG소프트 게임스쿨 수료 후 홀로 자취방에서 DirectX 3D 기반으로 빈프로젝트에 구현해본 3D 당구 시뮬레이션.
GitHub 계정에 당시 소스를 push 해 봄.
ASE 3D mode parser, 기본 전반사 물리 계산식, 쿼터니언(사원수) 적용함.
